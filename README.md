# 📱 Social Links Profile

This is a solution to the [Social links profile challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/social-links-profile-UG1OI9b8-G). Built with a focus on modern development standards, type safety, and responsive design.

## 🚀 Live Demo

## 🛠️ Tech Stack

- **React 19** - Modern UI library.
- **TypeScript** - For robust type-checking and developer experience.
- **Tailwind CSS v4** - Utilizing the latest engine for styling with CSS variables.
- **Vite** - High-performance build tool.

## ✨ Key Features & Learning Outcomes

- **TypeScript Architecture:** Implemented `UserProfile` and `SocialLink` interfaces to ensure data integrity across the component tree.
- **Tailwind v4 Configuration:** Leveraged the new `@theme` block in CSS to manage design tokens (colors, spacing, and typography) without a heavy JS config file.
- **Modular Component Design:** Clean separation of data and presentation logic.
- **Mobile-First Workflow:** Optimized for 375px mobile screens up to 1440px desktop displays.
- **Hover States & Transitions:** Smooth interactive elements using Tailwind's utility classes.
